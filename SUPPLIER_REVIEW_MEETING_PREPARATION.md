## 会议议程规划

### 第一部分：整体绩效回顾 (30分钟)
1. **供应商整体表现概述**
   - 关键绩效指标(KPI)总结
   - 同比/环比分析
   - 行业基准对比

2. **重点问题识别**
   - 质量问题汇总
   - 交付延误分析
   - 成本异常情况

### 第二部分：个别供应商评审 (90分钟)
1. **A级供应商评审** (优秀供应商)
   - 表现亮点总结
   - 战略合作深化机会
   - 产能扩展计划

2. **B级供应商评审** (一般供应商)
   - 改进要求明确
   - 支持计划制定
   - 监督机制建立

3. **C级供应商评审** (问题供应商)
   - 问题根因分析
   - 整改时间表
   - 替代方案准备

### 第三部分：改进计划制定 (45分钟)
1. **短期改进措施** (1-3个月)
   - 紧急问题解决方案
   - 快速改进项目

2. **中期发展计划** (3-12个月)
   - 供应商能力提升计划
   - 合作模式优化

3. **长期战略规划** (1-3年)
   - 供应商生态建设
   - 创新合作项目

## 评审标准与工具

### 供应商评分卡
| 评估维度 | 权重 | 评分标准 | 得分 | 备注 |
|---------|------|----------|------|------|
| 质量管理 | 30% | 合格率≥99%: 优秀<br>95-99%: 良好<br>90-95%: 一般<br><90%: 差 | | |
| 交付管理 | 25% | 准时率≥98%: 优秀<br>95-98%: 良好<br>90-95%: 一般<br><90%: 差 | | |
| 成本控制 | 20% | 成本节约≥5%: 优秀<br>0-5%: 良好<br>0-(-3)%: 一般<br><-3%: 差 | | |
| 服务水平 | 15% | 响应时间≤4h: 优秀<br>4-8h: 良好<br>8-24h: 一般<br>>24h: 差 | | |
| 创新能力 | 10% | 主动提出改进建议: 优秀<br>配合改进: 良好<br>被动执行: 一般<br>抗拒改进: 差 | | |

### 决策矩阵
- **90-100分**: 战略合作伙伴 - 深化合作，增加订单份额
- **80-89分**: 优选供应商 - 维持合作，适度激励
- **70-79分**: 合格供应商 - 继续合作，重点改进
- **60-69分**: 观察供应商 - 限制合作，强制改进
- **<60分**: 淘汰供应商 - 逐步替换，终止合作
